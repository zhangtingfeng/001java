供应链金融三方协议模板

甲方（融资方）：

乙方（平台方）：

丙方（保管方）：

   

甲、乙、丙三方就甲方将其货物储存于丙方仓库，丙方为甲乙双方提供货物的入库、出库、储存、保管服务等事宜，三方依据平等、自愿的原则，协商一致，遵照相关法律法规约定，达成如下协议。


第一条   权责关系

三方一致确认，对其储存于丙方仓库的所有货物，乙方向甲方采购后，乙方拥有所有权，甲方拥有支配权，甲方（或乙方）负责货物销售，乙方不得销售或挪用货物；甲方调配货物，需要乙方认证登记，方能办理出库手续。任何情况下丙方对上述货物均不享有任何形式的所有权和支配权。

协议期内，冷库使用费用由甲方代理乙方向丙方支付。

第二条   丙方的主要义务及服务范围

1.   丙方须应甲乙双方的要求为甲乙双方提供本协议约定的货物的仓储服务，包括但不限于入库、出库、保管等。

2.   丙方保证其对货物的仓储地点拥有完全的、排他的使用权或者所有权，不存在任何法律上的瑕疵。

3.   丙方应向甲乙双方提供完整的货物储存及出入库记录。丙方应配合乙方，提供纸质《库存确认单》并盖章确认。

4.   三方建立唯一库存业务微信群，对于货物出入库需求必须经过线下确认签字后拍摄上传微信群作为凭证，具有法律效益。

第三条   货物的保管条件和要求

1.   仓储保管条件：丙方应根据货物情况和协议约定，提供合格且安全可靠的仓储条件。

2.   仓储保管要求：丙方应按下列要求妥善保管甲乙方的货物。

(1) 应当采取措施防止甲方的货物在仓储过程中发生损耗，实际储存货物与系统统计数据误差应小于_ %

(2) 按照乙方平台仓的合作指导要求搭建智能、完善的智能仓储系统，并将库存管理系统权限移交至乙方，包括出入库管理、移库管理、库存监察等；

(3) 建立科学、严谨的仓储保管制度，对于货物的接收、储存、出库三个环节派专人严格监控、把关；

(4) 建立完整的档案资料和货物库存清单报表，及时与甲方核对库存账目，报告货物动态，及时准确提供货物接收、储存和出库报表；

(5) 应严格执行国家安全和消防管理规程，规范操作，保证甲方货物的安全；

(6) 应对甲方或乙方派出的盘库人员的工作，积极配合、协助和支持；

(7) 应定时对仓储货物进行检查，发现货物有异状，应及时通知甲方。

(8) 应对货物进行批次管理，先进后出。

3.   储存易燃、易爆、有毒、有腐蚀性、有放射性等危险物品或者易变质物品，甲方应当在向丙方发送《入库单》时一并书面说明该物品的性质，提供有关资料。违反前款规定的，丙方可以拒收甲方的货物，也可以采取相应措施以避免损失的发生，因此产生的费用由甲方承担。

4.   丙方发现甲方的货物有变质或者其他损坏情况的，危及其他仓储货物的安全和正常保管的，应当催告甲方与乙方作出必要的处置。因情况紧急，丙方可以作出必要的处置，但事后应当将该情况及时通知甲方，因此产生的费用由甲方承担。

第四条  货物的入库和验收（补货）

1.  甲方应在货物入库前向丙方发送《入库单》，知会乙方，并通知丙方做好接验货准备。如货物运达丙方仓库时丙方尚未收到任何甲乙双方的相关到货信息，则丙方有权拒绝接收货物。

2.  仓储货物由甲方或者甲方指定的第三方负责运送到库，丙方应按本协议约定的条件、时间、数量及时有效的接收货物。

4.  甲方向丙方入库的仓储货物应为包装完好、外观无污损、无淋湿（包装标准按国家或行业标准执行）。如没有以上标准的，由此造成的货物损失丙方不承担任何责任。

5.  货物入库时，丙方按照协议约定的货物的品名、规格、数量、外包装状况进行现场验收。货物包装内的货物品名、规格、数量，应以外包装或者货物上的标记为准；外包装或者货物上无标记的，以乙方提供的货物资料为准，乙方应保证货物资料的真实性。如发现入库货物与货物《入库单》或协议约定不符，丙方应及时通知甲乙双方；同时甲方也应及时给予丙方明确的处理指示，否则由此产生的损失由甲方负责。

7.  货物入库后，丙方应当场向甲乙双方出具《入库单》。

第五条   货物的出库确认

1.   货物出库时，甲方销售单以线上形式向乙、丙方发送。乙方确认后，丙方应凭销售单（载明的提货车号或提货人、货物名称、数量、规格以及其他载明条件）办理出仓手续。

2.   甲方应在线上发出后 3个工作日（或天）内将联合签字的《出库单》原件交与乙方、丙方各一份。

3.   货物出仓作业时丙方应小心谨慎，精心操作，以避免损毁货物。如遇货物有货损、货差、水湿、污染等情况时，丙方应做好货物的货损、货差、水湿、污染等记录，同时积极采取防止状况恶化的有效措施，并及时通知甲方。

4.   若提货方因货物品质或数量异议等原因拒绝提取货物，丙方应立即通知甲、乙方。若提货方未在甲方通知丙方的指定期限内提货，丙方应最迟于该期限届满的次日通知甲方。

第六条   违约责任

1.   丙方应保证《库存确认单》、《入库单》、《出库单》所载信息的准确性，若货物实际状况与丙方确认的上述凭证所载信息不符，丙方须承担赔偿实际损失的责任。

2.   在货物储存、保管期间，未按国家有关规定或者协议约定的要求操作、保管和储存货物的，造成货物发生损坏、灭失、变质、损耗、短少和污染的，丙方应当向甲方承担赔偿实际损失的责任；同时丙方应及时将货物的损坏、灭失、变质、损耗、短少和污染的情况及时通知甲方，否则应承担甲方因知情不及时而造成的其他实际损失的责任。

3.   储存期间，因丙方保管不符合本协议约定而造成仓储物毁损、灭失的，丙方应当承担赔偿实际损失的责任。因甲方的仓储货物性质、包装不符合协议约定或者超过有效储存期造成仓储货物变质、损坏的，丙方不承担任何赔偿责任。

4.   在货物的储存期间，货物的风险由丙方承担；如果货物出现损毁、灭失、等情况的，则由丙方承担损害赔偿责任，战争、自然灾害等不可抗力的情况除外。

5.   丙方保证其仓库内所有货物均为特定化管理，且任何对甲方货物的私自处分、质押等都是无效的。

第七条   协议生效

1.   本协议一式叁份，甲、乙、丙三方各执壹份，具有同等法律效力。

2.   本协议自三方签字并盖章之日起生效。本协议的有效期为            ，如协议到期三方均无提出解除协议，则视为三方同意按原协议条款约定继续执行三方的权利义务；如一方有异议，可另签新的协议。

3.   在协议履行过程中所使用的传真件为有效文件，但应按本协议约定的要求及时将传真件之原件交予对方。
