汇农撮合采购合同模板

甲方（采购方）：                            

乙方（供应商）：                           

甲乙双方经过汇农平台的撮合引导，本着可持续化共赢发展的目标，经沟通确认声明：双方均有资质和能力签署、履行本合同，双方在签订合同时均秉持公平、公正、诚信、务实的合作立场；如合同签署后任何一方有证据证明另一方在商讨签署本合同的过程中存在隐瞒、欺诈、虚构、夸大事实或利用非道德利益交换等行为促成本合同签订的，发现方可立即终止或部分终止本合同，同时可要求违反合同的一方赔偿因签署、履行本合同所产生的所有费用（包括但不限于已支付的定金、保证金、合同款等）。

双方根据《中华人民共和国合同法》及其他法律、法规的相关规定、《汇农平台服务协议》，在平等、自愿、诚信的基础上，经友好协商，甲乙双方根据采购货物事宜明确各自的权利义务，两方需共同遵守且执行。

1、采购内容

1.1甲方向乙方采购苹果，采购的货物种类、质量、数量、价格、交付时间、加工包装等以甲乙双方协商制定的《货物采购确认单》（见附件一）为准。

2、甲方权利义务

2.1甲方应保证向乙方提出的采购需求真实、准确，若因甲方提供的货物单证、资料与货物的实际情况不符及甲方其他原因而导致乙方遭受任何损失的，甲方应负责全额赔偿。

2.2 甲方无正当理由拒绝接收货物的，须承担乙方因此所发生的所有费用并赔偿乙方损失。

2.3甲方应按照本合同的约定， 在约定时间内支付货款，若甲方未按照本合同的规定及时支付有关款项，应按逾期支付款项的       %/日的标准向乙方支付滞纳金；若甲方经乙方通知后仍不支付的，乙方有权利自行处理货物。

3、乙方权利义务

    3.1乙方依照税法相关规定，并根据货物确认单开具增值税发票给甲方。

3.2乙方不能按照合同约定交货的，根据《合同法》规定，乙方应返还甲方两倍数额的定金。

3.3 乙方所交货物不符合规定的，如果甲方同意收取，应当按质论价；如果甲方不同意收取，应根据货物的具体情况，由乙方负责换货或返工或退货并承担因此而支付的实际费用。

5、违约责任

    5.1甲方拒绝履行本合同下的义务而导致乙方不能对外履行或提供本合同约定服务的，由此产生的一切损失由甲方全部承担。

5.2如一方违约，应当承担守约方为实现债权而支付的费用，包括但不限于催收费、诉讼费（仲裁费）、保全费、公告费、执行费、律师费、差旅费及其他费用。

6、商品运输

6.1运输方式：£甲方自提 £乙方运送。

6.2退货方式：协商后由过错方承担。

7、合同的变更和解除

    7.1本合同履行期间，经协商一致，双方可以变更本合同的相关条款。

7.2出现以下情形之一，双方可以解除或提前终止本合同：

（1）各方协商一致。

（2）法律规定或者本合同约定的其他可以终止合同的情形。

7.3双方按合同约定提前终止合同的，并不免除合法依据本合同以及各方约定应当承担的未履行的义务。

8、诚信及保密义务

    8.1本合同任何一方均有义务保守对方商业机密，未经对方的事先书面同意，不得向任何第三方披露对方商业机密，亦不得将该等商业机密用于履行本合同之外的其他目的。但根据有关政府机关或法院要求透露的不在此限，保密义务的有效期限为本合同签署之日起至权利义务履行完毕止，以及此后的__年。

8.2任何一方违反以上“诚信和保密条款”之约定的，守约方有权立即终止或部分终止本合同，并向违约方追偿损失。

10.争议的解决

10.1在履行本合同的过程中，若发生争议，双方应协商解决，协商不成的， 任何一方均有权向汇农平台发起纠纷处理申请，若仍对汇农平台处理意见有异议，任何一方均有权向合同签约所在地有管辖权的人民法院提起诉讼。

11、不可抗力

    11.1本合同所述之不可抗力系指政策变化、自然灾害等发生不可避免、后果无法克服的客观情况。

11.2任何一方因不可抗力而无法履行本合同的，将不视为违约，但该方应尽其所能减少另一方可能因此遭受的损失。

12、合同的生效及其他

    12.1本合同为电子合同，双方签署之日起具有法律效力。

12.2本合同未尽事宜由各方协商解决或签订补充合同，补充合同与本合同具有同等法律效力，补充合同与本合同不一致的以补充合同为准。
